# 📊 Data Analytics Project 2 – Exploratory Data Analysis (EDA)

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on an E-Commerce Sales Dataset. The objective is to analyze the dataset, identify patterns, trends, and outliers, and summarize key business insights using Microsoft Excel.

## 🎯 Objective
- Understand the dataset using descriptive statistics.
- Identify trends and patterns in sales.
- Detect outliers and missing values.
- Create charts and summaries for better visualization.
- Generate meaningful business insights.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts (Bar Chart, Pie Chart)
- Descriptive Statistics

## 📂 Dataset Information
- Total Records: 1200
- Total Columns: 14
- Dataset Type: E-Commerce Sales Data

## 📈 Analysis Performed
- Data Inspection
- Basic Statistics
- Product-wise Sales Analysis
- Order Status Analysis
- Payment Method Analysis
- Data Visualization
- Key Insights
- Conclusion

## 📊 Visualizations
- Bar Chart (Product Distribution)
- Pie Chart (Product Share)

## 🔍 Key Insights
- Electronics products have the highest number of orders.
- Most orders are successfully delivered or shipped.
- The dataset contains no significant missing values.
- A few high-value transactions were identified as outliers.

## ✅ Conclusion
The dataset is clean and suitable for analysis. Exploratory Data Analysis helped identify customer purchasing patterns, product performance, and overall sales trends. These insights can support better business decision-making.
